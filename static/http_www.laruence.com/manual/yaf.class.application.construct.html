名称

Yaf_Application::__construct

(Since Yaf 1.0.0.0)

public void Yaf_Application::__construct(mixed $config,
                                          string  $section = ap.environ );

初始化一个Yaf_Application, 如果$config是一个INI文件, 那么$section指明要读取的配置节.

参数
config

关联数组的配置, 或者一个指向ini格式的配置文件的路径的字符串, 或者是一个Yaf_Config_Abstract实例

返回值

void

例子

例 11.1. Yaf_Application::__construct的例子

     
     <?php
     $config = array(
      "ap" => array(
        "directory" => "/usr/local/www/ap",
       ),
     );
     $app = new Yaf_Application($config);
     ?>
输出 
     
     object(Yaf_Application)#1 (6) {
     ...
     }
